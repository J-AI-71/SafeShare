Schnellhilfe

Link wird nicht erkannt

  1. Bitte komplette URL einfügen (https://…).
  2. Leerzeichen vor/nach dem Link entfernen.
  3. Bei Textnachrichten nur den Link kopieren.

Ergebnis bleibt unverändert

  1. Prüfen, ob der Link überhaupt Parameter enthält (?).
  2. Seite neu laden und erneut versuchen.
  3. Bei Bedarf Browser wechseln (Safari/Firefox/Chrome).

Seite lädt nicht korrekt

  1. Seite mit Cache-Refresh neu laden.
  2. Browser-Erweiterungen testweise deaktivieren.
  3. Danach erneut testen.

Häufige Fragen

Was entfernt SafeShare?

SafeShare entfernt typische Tracking-Parameter wie utm_*, gclid oder fbclid. Ziel ist ein sauberer Link ohne unnötigen Tracking-Ballast.

Warum bleiben manchmal Parameter im Link?

Nicht jeder Parameter ist Tracking. Einige Werte sind technisch nötig (z. B. Sprache, Seite, Produkt-ID) und bleiben deshalb erhalten, damit der Link korrekt funktioniert.

Werden meine Links gespeichert oder hochgeladen?

Nein. Die Bereinigung läuft lokal im Browser. Es werden keine eingegebenen Links auf einem Server gespeichert.

Kann ich SafeShare für Messenger, E-Mail und Social Media nutzen?

Ja. Du kannst gereinigte Links in WhatsApp, Signal, E-Mail, Newslettern oder Social-Posts verwenden.

Funktioniert SafeShare auch mobil?

Ja. SafeShare funktioniert auf Smartphone, Tablet und Desktop direkt im Browser.
