Quick help

Link is not detected

  1. Paste the full URL (https://…).
  2. Remove spaces before/after the link.
  3. If copied from a message, copy only the link.

Result stays unchanged

  1. Check whether the URL has parameters (?).
  2. Reload and try again.
  3. Try another browser (Safari/Firefox/Chrome).

Page does not load correctly

  1. Do a hard refresh / cache refresh.
  2. Temporarily disable browser extensions.
  3. Test again.

Frequently asked questions

What does SafeShare remove?

SafeShare removes common tracking parameters such as utm_*, gclid, and fbclid. The goal is a clean link without unnecessary tracking noise.

Why do some parameters remain?

Not every parameter is tracking. Some are required for functionality (e.g. language, page, product ID), so they stay to keep the link working.

Are my links stored or uploaded?

No. Cleaning runs locally in your browser. Entered links are not stored on a server.

Can I use SafeShare for messenger, email, and social posts?

Yes. You can use cleaned links in WhatsApp, Signal, email, newsletters, and social posts.

Does SafeShare work on mobile?

Yes. SafeShare works on smartphone, tablet, and desktop directly in the browser.
