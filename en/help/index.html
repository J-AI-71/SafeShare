Quick help

Link is not detected

  1. Paste the full URL (https://…).
  2. Remove spaces before/after the link.
  3. If copied from a message, copy only the link.

Result stays unchanged

  1. Check if the URL has parameters (?).
  2. Reload the page and try again.
  3. If needed, test another browser (Safari/Firefox/Chrome).

Page behaves oddly

  1. Do a hard refresh.
  2. Temporarily disable browser extensions.
  3. Try once more.

Frequently asked questions

What does SafeShare remove?

SafeShare removes common tracking parameters such as utm_*, gclid, and fbclid to produce cleaner links.

Why do some parameters remain?

Not every parameter is tracking. Some keys are functional (for language, page, or product context) and may be kept so links still work correctly.

Are my links uploaded or stored?

No. Cleaning runs locally in your browser. Entered links are not uploaded to a server.

Can I use SafeShare for messenger, email, and social media?

Yes. Clean links can be shared in WhatsApp, Signal, email, newsletters, and social posts.

Pro does not start?

Unzip the package first, then open 01-START.html in your browser.
