Bookmarks / Bookmarklets

Bookmarklets are bookmarks that run a small script. On desktop, they let you use SafeShare very fast: take the current URL → open SafeShare → optionally clean immediately.

Setup (desktop)

  1. Create a bookmark.
  2. Name: e.g. SafeShare / SafeShare 1 click.
  3. URL field: paste the bookmarklet code (must start with javascript:).
  4. Click it on any website → SafeShare opens.
Tip: paste it as one single line (no line breaks). Drag buttons are even more reliable.

Troubleshooting

Nothing happens

  • Check: does the bookmark URL really start with javascript:?
  • Was the code truncated/rewritten? (some browsers do that)
  • Test on a “normal” website (not a PDF viewer / not internal browser pages).

SafeShare opens, but the field is empty / run=1 doesn’t start

  • Then it’s almost always cache (an older app/script version).
  • Test the app directly: /en/app/?u=https%3A%2F%2Fexample.com%2F%3Futm_source%3Dx&run=1
