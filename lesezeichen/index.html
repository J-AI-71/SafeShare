Lesezeichen (Bookmarklets)

Desktop: ein Klick, Link sauber.

Ziehe einen Button in deine Lesezeichenleiste. Danach kannst du auf jeder Seite den aktuellen Link bereinigen und direkt kopieren oder in SafeShare öffnen.

Installation (Desktop)

  1. Lesezeichenleiste im Browser einblenden.
  2. Den gewünschten Button per Drag & Drop in die Leiste ziehen.
  3. Auf beliebiger Seite auf das Lesezeichen klicken.
iOS/iPadOS: Bookmarklets sind oft unzuverlässig. Nutze dort besser den Shortcut-Flow.

Clean + Copy (Code)

javascript:(()=>{const u=new URL(location.href);const DROP=new Set(['gclid','dclid','gbraid','wbraid','fbclid','msclkid','igshid','twclid','ttclid','li_fat_id','yclid','mc_cid','mc_eid','mkt_tok','_hsenc','_hsmi','srsltid','obclid','tblci','_ga','_gid','_gac','_fbp','_fbc']);const p=u.searchParams;[...p.keys()].forEach(k=>{const kl=k.toLowerCase();if(kl.startsWith('utm_')||DROP.has(kl))p.delete(k)});u.search=p.toString()?('?'+p.toString()):'';const out=u.toString();const done=()=>alert('SafeShare: Link bereinigt und kopiert');if(navigator.clipboard&&navigator.clipboard.writeText){navigator.clipboard.writeText(out).then(done,()=>prompt('Kopieren:',out));}else{prompt('Kopieren:',out);}})();
